CalMoyenne

CalMoyenne est une extension Firefox qui calcule automatiquement vos moyennes sur les sites de notes scolaires.

Télécharger l'extension

Fonctionnalités

  • Calcul automatique des moyennes
  • Détection intelligente des notes
  • Gestion des coefficients par classe
  • Interface intuitive
  • Mise à jour automatique des calculs

Installation

  1. Cliquez sur le bouton "Télécharger l'extension" ci-dessus
  2. Firefox vous demandera l'autorisation d'installer l'extension
  3. Cliquez sur "Ajouter" pour installer CalMoyenne
  4. L'extension sera automatiquement activée
