Générateur d'Icônes PNG pour l'Extension

Cet outil vous permet de créer les fichiers PNG nécessaires à partir de votre icône SVG.

16x16


48x48


128x128


Instructions

  1. Assurez-vous que le fichier icons/icon.svg existe dans votre dossier d'extension
  2. Ouvrez cette page dans votre navigateur
  3. Cliquez sur les boutons pour télécharger les icônes PNG
  4. Placez les fichiers téléchargés dans le dossier icons/ de votre extension
